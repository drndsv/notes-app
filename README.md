# NotesApp

## Установка:

```bash
npm ci
```

## Запуск

Чтобы запустить локально:

```bash
ng serve
```

Приложение запустится по адресу: `http://localhost:4200/`

## Запуск сервера:

```bash
npm run server
```

Сервер запустится по адресу: `http://localhost:3000/`

## Запуск линтера и форматирования

### Запуск линтера:

```bash
npm run lint:fix
```

### Проверка линтером:

```bash
npm run lint
```

### Запуск форматирования:

```bash
npm run format
```

### Проверка форматирования:

```bash
npm run format:check
```
